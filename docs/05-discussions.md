Discussions
Каждая позиция содержит вкладку «Обсуждения». Сообщения в обсуждениях включают: имя автора, временную метку, текстовое содержимое (в формате Markdown).

Если страницу просматривают рекрутеры, имя автора ведет на общедоступный профиль пользователя.

Сообщения отображаются в хронологическом порядке. Новые сообщения всегда добавляются в конец, сообщения нельзя вставлять между существующими. Обновления должны появляться для всех активных пользователей в течение 2–5 секунд (реализация может использовать WebSockets, опросы или что-то подобное).
