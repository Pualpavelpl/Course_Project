Разработать веб-приложение для системы управления резюме (учет различных должностей, опыта работы, навыков и т. д.). Пользователи создают описания «должностей», задавая для них набор атрибутов. Должность выступает в роли шаблона для резюме; другие пользователи заполняют свои резюме для конкретных должностей, указывая соответствующие данные. К числу наиболее важных функций относятся: возможность настройки должностей с произвольным набором атрибутов, использование библиотеки атрибутов для многократного применения и автоматическая генерация резюме.

Например, рекрутер создает должность «Бизнес-аналитик», добавляет атрибут «Уровень владения английским языком» (выбор из списка) и числовое поле «Средний балл» (GPA). Кандидаты указывают уровень английского и средний балл в своих профилях и формируют резюме, адаптированные под данную должность. Затем рекрутеры просматривают полученные резюме, чтобы подобрать подходящего кандидата.

Система представляет собой веб-платформу для подбора персонала, позволяющую кандидатам вести профессиональные профили многократного использования и формировать адаптированные резюме для вакансий, которыми управляют рекрутеры.

Система представляет собой веб-платформу для подбора персонала, позволяющую кандидатам вести профессиональные профили многократного использования и формировать адаптированные резюме для вакансий, которыми управляют рекрутеры.

В основе платформы лежат три ключевые функции:
Библиотека атрибутов многократного использования — атрибуты можно определить один раз и использовать для различных вакансий и резюме;
Настраиваемые шаблоны вакансий — рекрутеры могут создавать шаблоны резюме, специфичные для конкретных вакансий, используя атрибуты из библиотеки;
Автоматическая генерация резюме — резюме формируются динамически на основе данных профиля кандидата и требований вакансии.
